# 📚 Online Book Store Sales Analysis using PostgreSQL
## Project Overview
This project demonstrates SQL skills by analyzing an Online Book Store database using PostgreSQL.
The project covers database creation, table relationships, SQL querying, aggregation, joins, and business insights.
It is designed as a beginner-to-intermediate SQL portfolio project.
---
## Database Tables
The project consists of three relational tables:
- Books
- Customers
- Orders
Relationships:
Books (1) ------< Orders >------ (1) Customers
